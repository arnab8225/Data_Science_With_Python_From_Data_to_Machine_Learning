## 🚀 Project Overview

Customer churn is a critical problem in the telecom industry. Retaining existing customers is more cost-effective than acquiring new ones.

This project aims to:
<ul>
<li>Analyze customer behavior</li>
<li>Identify churn patterns</li>
<li>Build a predictive model to forecast churn</li>
</ul>

---

## 🧠 Machine Learning Workflow

<b>📌 Data Collection</b>
<ul>
<li>Dataset: Telco Customer Churn Dataset</li>
<li>Loaded using pandas</li>
</ul>

<b>📌 Data Preprocessing</b>
<ul>
<li>Removed unnecessary columns (customerID)</li>
<li>Handled missing values</li>
<li>Converted categorical data using Label Encoding</li>
</ul>

<b>📌 Feature Engineering</b>
<ul>
<li>Encoded categorical features</li>
<li>Balanced dataset using SMOTE</li>
</ul>

<b>📌 Model Training</b>
<ul>
<li>Algorithm: XGBoost Classifier</li>
<li>Train-test split applied</li>
</ul>

<b>📌 Model Evaluation</b>
<ul>
<li>Accuracy Score</li>
<li>Confusion Matrix</li>
<li>Classification Report</li>
</ul>

<b>📌 Model Saving</b>
<ul>
<li>Model saved using joblib</li>
</ul>

## 📊 Key Insights

<ul>
<li>Contract type and tenure strongly influence churn</li>
<li>Customers with monthly contracts are more likely to churn</li>
<li>Higher monthly charges increase churn probability</li>
</ul>
